## 1. サイト全体構成（サイトマップ）

グローバルナビは**5つの親カテゴリ＋2つの単独項目**で構成。親カテゴリはドロップダウン（SPではアコーディオン）。

| カテゴリ（英語＋日本語の2段表記） | 配下ページ |
|---|---|
| TOP トップ | `/` |
| ABOUT walea（waleaについて） | CONCEPT こだわり `/concept/`・STAFF スタッフ紹介 `/staff/`・MEDIA メディア掲載 `/media/` |
| NAIL DESIGN（ネイルデザイン） | NEW COLLECTION 最旬特集 `/collection/`・WORKS お客様ネイル `/works/`・CAMPAIGN `/campaign/` |
| GUIDE（初めての方へ） | FLOW 予約から施術の流れ `/flow/`・Q&A よくある質問 `/faq/`・GEL NAIL ジェルネイルとは `/gelnail/` |
| MENU（メニュー） | NAIL&CARE メニュー `/menu/`・WEDDING NAIL `/weddingnail/` |
| SALON&ACCESS サロン&アクセス | `/access/` |
| BLOG&SNS ブログ&SNS | `/sns/` |

フッターのみの補助ナビ：NEWS `/news/`・SITEMAP・RECRUIT・PRESS。
予約導線：CONTACT ご予約・お問い合わせ `/contact/`（ナビとは別の常設ボタン扱い）。

### 制作版（架空サロン）への落とし込み
計画の制作範囲「TOP＋メニュー／スタッフ／ご予約・アクセス」に合わせて以下に集約する：

- `index.html` … TOP
- `menu.html` … ネイル&ケアメニュー（＋初回特典）
- `staff.html` … コンセプト＋スタッフ紹介（ABOUT系を統合）
- `access.html` … サロン案内・アクセス・予約までの流れ・予約/問い合わせ（GUIDE＋SALON&ACCESS＋CONTACTを統合）

## 2. 共通レイアウト

### 2-1. ヘッダー（PC）
- 上段右：電話番号（`tel:`リンク・アイコン付き）＋「ご予約/お問い合わせ」ボタン（アクセントカラーのミント地・白文字）
- 中央：筆記体風ロゴ（画像）。ロゴ下にグローバルナビを横一列配置
- ナビ項目は「英語（Raleway）＋日本語小文字」の2段表記、ホバーでドロップダウン展開

### 2-2. ヘッダー（SP・375px）
- `position: fixed` の追従ヘッダー（高さ**57px**）：左にロゴ、右に「予約/お問い合わせ」短縮ボタン＋ハンバーガー
- ハンバーガーで全画面ドロワーメニュー（カテゴリはアコーディオン展開）
- ページ下部追従バーは無し（ヘッダー内に予約ボタンを常設する方式）

### 2-3. フッター（全ページ共通）
1. **RECOMMEND CONTENTS**：おすすめ3カード（NEW COLLECTION／CAMPAIGN／WORKS）。画像＋英語見出し＋日本語ラベル
2. **フッター本体**（ダーク系背景）：ロゴ・SALON&ACCESSリンク・SNSアイコン（Instagram/Facebook/Twitter）・電話ボタン・LINEボタン・住所・Google Mapリンク
3. 補助ナビ（NEWS/SITEMAP/RECRUIT/PRESS）＋コピーライト

### 2-4. 下層ページ共通フレーム
- ページ先頭に **mainTitleBox（高さ約112px）**：パンくず＋H1（英語大見出し＋日本語サブタイトル）
- コンテンツは `contentWrapper` に格納、末尾に「PAGE TOP」リンク → RECOMMEND CONTENTS → フッター

## 4. 下層ページの構成

### 4-1. MENU（ネイル&ケアメニュー）— 約8,000px
- カテゴリ大分類は **HAND / FOOT / OTHERS** の3つ。各分類の先頭にダーク地（#2B2E38系）の帯見出し（H2）
- 分類内は H3 の中分類（ソフトジェル／スカルプチュア／ケア&カラー／ハンドマッサージ・パック等）ごとに価格表
- 価格表の行：メニュー名＋説明文（小さめグレー）＋右端に税込価格。一部に参考写真サムネイル
- 末尾に「ジェルネイル安心保証制度」説明枠と初回特典の再掲

### 4-2. STAFF（スタッフ紹介）— 約1,100px
- 導入文（サロンの技術력・資格のアピール）
- スタッフカード×2：円形写真（左）＋右側に情報テーブル
  - 項目はタグ風ラベルで「歴」「資格」「得意」＋「Message」（ひとことコメント）
  - 名前は「ネイリスト＋漢字名＋ローマ字」の3点セット

### 4-3. FLOW（ご予約から施術の流れ）— 約5,000px
- **01〜06の番号付きステップカード**（ご予約→ご来店→初回手続き→カウンセリング・施術→お会計→アフターケア）
- カード間をアクセントカラーの下向き三角形で接続（縦タイムライン）
- ステップ04内に「各メニューの流れ」としてメニュー別（ジェル/スカルプ/ケア/フット）のサブカード
- 05に取扱クレジットカードのブランドアイコン一覧

### 4-4. SALON&ACCESS — 約2,200px
- 店内写真（大）→ サロン情報テーブル：ADDRESS／TEL/FAX／MAIL／OPEN TIME（曜日別の最終受付も記載）／HOLIDAY
- 支払い方法（カードブランドアイコン）、駐車場、サイトURL、スタッフ数などの補足表
- Google Mapへのテキストリンク（埋め込み地図ではない）

### 4-5. CONTACT（ご予約・お問い合わせ）— 約2,300px
- 電話予約ブロック（ダーク帯見出し＋電話番号＋受付時間）
- メールフォームブロック：仮予約フォーム（テーブルレイアウト）
  - 項目：利用回数（ラジオ）／希望メニュー（チェックボックス）／希望日時（第1〜3希望・日付＋時刻）／氏名／ふりがな／電話／メール／コメント
  - 必須項目に赤系「必須」タグ。送信→確認画面の2段階

### 4-6. CONCEPT（こだわり）— 約3,400px ※staff.htmlに統合予定
- 01 Design／02 Skill／03 Space／04 Service の**番号付き4ブロック**
- 各ブロック＝大きな番号＋英語見出し＋日本語タイトル＋写真＋H3の小見出し2〜4個＋本文

### 4-7. その他（参考のみ・制作範囲外）
- WEDDING NAIL：リード→プラン表→ジェル/オーダーチップ比較→おすすめメニュー
- FAQ：カテゴリ別アコーディオンQ&A（5カテゴリ）
- NEWS：日付＋キャンペーン記事の単純リスト（年別アーカイブ）

## 5. デザイントークン（計算済みスタイルから抽出）

### 5-1. カラーパレット（使用頻度順）
| 色 | HEX | 用途 |
|---|---|---|
| テキスト基本色 | `#333333` | 本文・見出し |
| アクセント（ミントグリーン） | `#23C8A2` | ボタン・リンク・帯・三角形コネクタ・バッジ |
| アクセント明 | `#43DCBA` / `#36CEAC` | ホバー・グラデーション系 |
| ダーク | `#2B2E38` | フッター背景・メニューのカテゴリ帯 |
| 背景グレー | `#F5F5F5` | セクション背景・テーブルヘッダ |
| 白 | `#FFFFFF` | 基調背景 |
| 補助グレー | `#6D6D6D` / `#959595` | 説明文・キャプション |

※CSS変数（`:root`）は未使用。色は直接指定。制作版ではCSS変数化する（dental-sample方式）。

### 5-2. タイポグラフィ
| 要素 | 指定 |
|---|---|
| 本文 | システム系サンセリフ（-apple-system, Noto Sans Japanese, ヒラギノ角ゴ…）、**14px / line-height 28px（2.0）** |
| 英語見出し | **Raleway**（sans-serif）。H2: 26〜28px / weight 400〜500、H3: 16px / weight 600 |
| 装飾見出し | **Libre Baskerville**（serif）— CONCEPT見出し等。一部リードに游明朝 |
| ロゴ | 筆記体（画像として実装されている） |
| 見出しパターン | 「英語大見出し＋日本語小サブタイトル」の2段組が全サイト共通 |

### 5-3. レイアウト
- コンテンツ最大幅：**約1072px**（内側要素は1000px）
- 余白：セクション間は大きめ（80px前後）。白ベースで余白を贅沢に使う
- 円形写真リンク・ひし形キーワードなど、**丸・ひし形の幾何学モチーフ**がアクセント

### 5-4. レスポンシブ挙動
- SPはヘッダーがfixed（57px）＋ハンバーガードロワー
- 1カラム化、円形リンクは2列グリッド維持、テーブルは縦積みに再構成
- 画像は遅延読み込み

## 6. CTA設計（予約導線）

| 導線 | 配置 |
|---|---|
| 電話（`tel:`リンク） | ヘッダー右上・フッター・CONTACTページ |
| 「ご予約/お問い合わせ」ボタン（ミント地白文字） | ヘッダー常設（PC/SP両方）・フッター |

## 7. 制作版への示唆（ステップ2への引き継ぎ）

1. **構成は踏襲、ページ数は4枚に統合**（§1参照）
2. **配色はオリジナル化**：参考はミント×ダーク。制作版は計画通り「上品なベージュ×ゴールド系」等の代替案を提示し、ユーザーが選択
3. **共通パターンを再現**：英語＋日本語2段見出し／円形写真リンク／番号付きステップ／帯見出し価格表／初回特典バッジ
4. **CSS設計**：dental-sample同様、CSS変数・モバイルファースト・ブレークポイント768/1024
5. **フォント**：Google Fonts（英語＝Raleway系 or Cormorant等のserif、日本語＝Noto Sans JP＋見出しにNoto Serif JP）で再現可能
6. 写真はプレースホルダ（CSSグラデーション＋SVG）で構成し、後日差し替え可能な構造に
