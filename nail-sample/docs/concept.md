# 架空ネイルサロン「Lino」コンセプト設計書

> 制作計画ステップ2の成果物。ユーザー承認済み：サロン名 **Lino**／配色 **ベージュ×ゴールド**（2026-06-12）

## 1. サロン概要（すべて架空）

| 項目 | 内容 |
|---|---|
| サロン名 | **nail salon Lino**（ネイルサロン リノ） |
| 名前の由来 | ハワイ語「Lino」＝**輝く・まぶしい**。ジェルの艶と、指先から生まれる自信の輝きを表す |
| コンセプトコピー | **「指先から、わたしが輝く。」** |
| サブコピー | 南青山の隠れ家ネイルサロン。上質なケアと、あなたらしい輝きを。 |
| ターゲット | 30〜40代の働く大人の女性。トレンドより「上品さ・持ち・仕上がりの美しさ」を重視 |
| 架空所在地 | 東京都港区南青山1-2-3 リノビル2F（外苑前駅 徒歩4分） |
| 架空電話番号 | 03-0000-0000（ダミー表記） |
| 営業時間 | 平日 10:00–20:00／土日祝 10:00–19:00／水曜定休 |
| 強み（4本柱） | Design（上品な大人デザイン）／Care（爪を傷めない丁寧なケア）／Space（半個室の安らぎ空間）／Hospitality（心を込めた接客） |

## 5. サイトマップ（4ページ構成）

```
index.html   TOP …… ヒーロー／お知らせ／CONCEPT（4本柱）／PICK UP／MENU・STAFFへの円形リンク／FIRST GUIDE（初回特典）
menu.html    MENU …… HAND／FOOT／OTHERS別の価格表＋安心保証＋初回特典
staff.html   ABOUT …… コンセプト詳細（4本柱の深掘り）＋スタッフ紹介2名
access.html  RESERVE & ACCESS …… ご予約の流れ（5ステップ）＋サロン情報＋予約フォーム（ダミー）
```

## 6. 各ページのワイヤー概要

### TOP（index.html）
1. ヘッダー（PC：ロゴ＋ナビ＋電話＋予約ボタン／SP：固定ヘッダー＋ハンバーガー）
2. ヒーロー：全幅ビジュアル（CSSグラデーション＋装飾）＋コンセプトコピー
3. お知らせ1行バー
4. CONCEPT：リード文＋ Design/Care/Space/Hospitality の4キーワード
5. PICK UP：キャンペーンカード2枚（季節コレクション／初回特典）
6. 円形リンク：MENU／STAFF／ACCESS
7. FIRST GUIDE：初回特典バッジ（①オフ無料②20%OFF）＋流れへのボタン
8. フッター（おすすめ3カード＋ダーク帯フッター）

### MENU（menu.html）
ページタイトル帯 → カテゴリ帯見出し（HAND/FOOT/OTHERS）→ 中分類ごとの価格表（名前＋説明＋税込価格）→ 安心保証 → 初回特典再掲

### ABOUT（staff.html）
ページタイトル帯 → コンセプト4本柱（番号付きブロック）→ スタッフカード2名（円形写真プレースホルダ＋経歴・資格・得意・Message）

### RESERVE & ACCESS（access.html）
ページタイトル帯 → ご予約から施術の流れ（番号ステップ＋ゴールド三角コネクタ）→ サロン情報テーブル＋ダミー地図 → 電話予約ブロック → 予約フォーム（送信なしのダミー）

## 7. 写真の扱い

実写真は使用せず、**CSSグラデーション＋SVG装飾（葉・爪・キラめきモチーフ）のプレースホルダ**で構成。
`<img>` ではなく背景クラスで実装し、後日 `background-image` 差し替えだけで実写真に置換できる構造にする。
