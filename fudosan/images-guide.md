# クレアシティ 画像差し替えガイド

現在はUnsplash（商用利用可）の写真を直リンクで使用しています。
ChatGPT / Codex などで生成したオリジナル画像に差し替える場合は、
生成した画像をこのフォルダ内の `images/` に保存し、`index.html` の該当URLを
`images/ファイル名.jpg` に書き換えてください。

## 注意

- 生成画像に実在のビル名・企業ロゴ・ランドマーク（例：東京タワー等の特定建造物）が入らないようにしてください
- ファイルサイズは1枚500KB以下推奨。JPEGまたはWebP形式
- 差し替え後もフォールバック（読み込み失敗時のグラデーション表示）はそのまま機能します
