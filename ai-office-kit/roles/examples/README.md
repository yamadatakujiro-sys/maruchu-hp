# カスタム職種の見本（examples/）

このフォルダは **標準8職種には含めない「カスタム職種（有料オプション）」の見本**です。
顧客の業種に合わせて職種を追加注文されたときの、テンプレ作成のたたき台・営業時の品質サンプルとして使います。

- 標準ラインナップは `roles/` 直下の8職種（leader / lp / designer / writer / video / sns / researcher / analyst）で**固定**。
- ここの見本は「こういう専用AIも作れます」という提案材料。実案件では顧客の実データ（メニュー・在庫・予約システム等）を
  `《 》` プレースホルダに流し込んで仕上げる。
- `MEMBERS` に組み込む場合の書式は通常の職種と同じ：`"<dir>:<表示名>:<role-template.md>"`。

## 収録見本
- `salon-reception.md` … 美容サロン 予約・接客AI（LINE窓口で予約受付・変更・接客・再来店促進）
