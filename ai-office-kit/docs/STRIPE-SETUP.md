# Stripe決済 準備手順（直販＝LINE導線の受け皿）

> X→公式LINE→無料相談→成約 の「成約時に支払ってもらう」ための決済準備。
> ココナラ経由の販売は「ココナラの決済」を使う（規約遵守・外部誘導しない）。Stripeは**直販専用**。
> 価格（確定）：**初期構築 ¥39,800（一括）＋ 月額運用 ¥9,800（サブスク）**。AI(Claude)利用料は顧客負担で別。

## 方式
- **Stripe Payment Links（コード不要）** を使う（非技術者向け・LINEで相談成約後にリンクを送る）。
- 相談は手動チャットなので、**決済リンクを案内するタイミングは自分で管理**できる（先走って公開しなくてよい）。

## 手順
1. **Stripeアカウント登録**：区分＝個人事業主/個人、国＝日本、本人確認、入金用の銀行口座を登録。
2. **商品を2つ作成**（商品カタログ）：
   - 初期構築：¥39,800／**一括（1回）**
   - 月額運用：¥9,800／**継続（毎月・サブスク）**
   - 商品説明は正直に（**収益断定しない＝景表法**）。「AIが業務を補助」「成果保証なし」を明記。
3. **Payment Link を各商品で発行**（初期用・月額用）。相談成約後にLINEトークで案内。
   - 初期＋月額を1つのチェックアウトにまとめることも可（初回に一括+サブスク開始）。運用しやすい形を選ぶ。
4. **特商法表記** と **プライバシーポリシー** のページを用意（法令・Stripe規約上必要）。
   - Stripeの「顧客サポート/ビジネス情報」やチェックアウトにリンクを設定。ページはnote等でも可。

## ⚠️ 法務の注意（`LEGAL-QUESTIONS.md` の弁護士相談トピック）
- **特商法表記**：原則 氏名・住所・電話番号・販売価格・支払方法・提供時期・**返金/解約条件**・継続課金の条件を開示。自宅住所を出したくない場合の手当て（バーチャルオフィス等）は要検討。
- **プライバシーポリシー**：顧客・そのエンドユーザーの個人情報の扱い。
- **契約書**：顧客名義Claudeの代理運用、成果物・IP、秘密保持、免責、サービス停止時の扱い、サブスク条件。
- **景表法**：「収益が上がる/成果が出る」等の断定・保証はしない。
- → **取引開始前に一度、弁護士等へ確認する**（本書・LEGAL-QUESTIONSは論点整理であり法的助言ではない）。
