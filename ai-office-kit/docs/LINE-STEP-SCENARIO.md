## 設計の考え方（忘れない・全配信の土台）

### リストマーケの4ステップ／LINEの守備範囲
1. 認知・集客（X等が得意。LINEは弱い→「友だち追加」導線）
2. リスト化（友だち追加＝ここからLINEの本領）
3. 教育（ステップ配信で価値・必要性を段階的に）
4. 販売／行動促進（CTAで相談・申込へ）
→ **LINEが担うのは「リスト化以降（特に教育と販売）」**。集客はXで。

### 3つのシナリオ（全体の役割）
- **教育シナリオ**：見込み客に「良さそう」を作る。**メリットを積み上げる（ポジティブ方向）**。
- **セールスシナリオ**：迷い・不安を消し行動させる。**「やらない理由」を潰す（ネガティブを消す方向）**。
- アフターフォロー：購入後のリピート/紹介（今回はスコープ外）。
→ 教育で欲しい気持ち、セールスで不安解消、の両輪。**教育を省くと比較検討層は離脱**。

### 各メッセージに必ず3点セット
- **目的**：この1通で何を理解・感じ・行動してほしいか
- **CTA**：次の具体行動。**小学生でも分かる直球**（「『デモ』と送ってください」）。遠回しは離脱。
- **ユーザー心理**：受け取る瞬間の気持ち。**登録からの経過時間**で変わる（直後＝熱量MAX／数日＝登録理由は覚えてる／1週間＝冷め気味、不安を言語化して消す）。

### 初回オファーを“先”に出す
教育を全部終えてからオファー、ではなく、**登録直後（あいさつ）に初回オファー（デモ視聴）**を置く。
→ 顕在層を取りこぼさず、後続の教育が「見るか決めるための情報」として読まれ開封率が上がる。

### 採用したテクニック（元設計書より）
- **動画ファースト**：初回オファーは「デモを見る」という超軽いハードル（買わせる前に1歩）。
- **時刻指定配信**（登録N日後 19:00）：経過時間指定は深夜配信リスク → 必ず時刻指定。平日17〜20時が当たり。
- **1メッセージ＝1CTA**／**同日連投しない**。
- **2択**（今すぐ見る／あとで）：能動選択させ「無視」状態を作らない。
- **応答メッセージ（キーワード）**で「デモ／あとで／相談」を受ける。
- **感想収集**：感想を書く＝自己説得＋熱い見込み客の可視化。ハードルが高いので3通で切り口を変えて依頼。

### ★タツ版の方針（元講座との違い＝重要）
- 「月100万続出」等の**収益断定コピーは使わない**（景表法・誇大広告／著作権）。
- 「本人出演動画／感想フォーム／企画オファー」の情報商材ローンチ型ではなく、
  **感想収集 → 無料相談 → 申込（Stripe）** に着地させる（done-for-you サービスに自然な形）。
- **今はテキスト版で稼働**。リッチメッセージ画像・デモ動画は後日作成して差し替える
  （今は「デモ」キーワードで手動実演）。

## LINEでの設定方法
- **0番（あいさつ）**：ホーム →「あいさつメッセージ」に貼る（友だち追加直後に自動送信）。
- **1〜5番**：左メニュー **「ステップ配信」** → 新規作成 → シナリオ名（例「導入導線_v1」）→ 開始条件 **「友だち追加」**
  → 各メッセージを **配信タイミング＝時刻指定（登録N日後 19:00）→ 本文貼付** の順で追加 → 5通組んだら**有効化（公開）**。
  ※今はテキスト版（リッチメッセージ画像・動画なし）。CTAは本文内テキストで運用。
- **応答設定＝チャット（手動）**にして、「デモ／あとで／相談」の返信に個別対応。
  応答メッセージに上表のキーワードを登録（自動応答＋手動の併用）。
- 「相談」が来たら→ ヒアリング → 提案 → Stripeの支払いリンクを案内（成約）。
